    npm、cnpm和nvm

    STF目前使用的是8.16版本的Node环境，需要使用多版本Node。

    # 查看长期稳定版本
nvm ls-remote --lts
...
v10.17.0   (Latest LTS: Dubnium)
v12.13.0   (Latest LTS: Erbium)

#安装node 12版本
nvm install v12.13.0

    淘宝镜像安装appium server

    ZH：https://npm.taobao.org/ EN：https://cnpmjs.org/

    安装cnpm：npm install -g cnpm --registry=https://registry.npm.taobao.org 安装appium server：cnpm install -g appium 检查版本：appium -v
