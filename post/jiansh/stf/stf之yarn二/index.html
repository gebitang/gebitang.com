    第一次只注意到了 Done ，没注意到failed with 502，直接启动，会遇到Module not found: Error: Cannot resolve module 'angular' 这是因为bower没有安装完整前端需要到模块。

    参考Bower installation of Angular fails due to use of deprecated Bower registry on Heroku ，修改工程下到 .bowerrc 文件——

    {
 "directory": "res/bower_components",
  "registry": "https://registry.bower.io"
}

    还是可能出现其他安装错误，因为要从github验证并下载模块。 多试几次yarn install，最后出现树状输出则表示安装成功。
