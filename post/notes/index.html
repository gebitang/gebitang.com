《我和吴祖光》

起因是看了这篇《颠倒道歉》，第二次正好搜到了94年版本。看了一半目前——中间有几页纸张没裁开，相当于我是第一位看到这本书的这个位置的?——了解了更多关于文革的内容（那时到底发生了什么，讳莫如深反而更让人好奇，这书现在估计都无法发表了），释放“恶”的时代

跟溥仪一起干活，新凤霞还另外写了一本《我和溥仪》，PDF版本已找到。

《西方现代思想》得到课

刘擎·西方现代思想“来一场观念的探险”。每天路上听一听，先把正餐听完，每一课讲得深入浅出，听得惊心动魄。

每一位对思想，个人，社会，政治有兴趣、有好奇、有疑惑的人都应该来听这门课。可以对许多“观念”做一下“正本清源”，你以为的你这样想的理由也许不是你以为的理由。

以历史的脉络梳理一遍那些“你偶尔听过，觉得是那么回事，又不知道为什么是那回事，以及好像也不是那么回事”的“观念”。

所谓“惊心动魄”是因为每一讲的内容都会引起思考，探索。有些“观念”会受到冲击，更重要的是那些“这个问题我从来没这么想过”的观念，它们并不以新奇取胜，但却让你更理解这个所处的“时代”和“世界”。

思想如旧，常听常新。

决策思维

一本小册子，听flomo的少楠一句话推荐的：“决策就是分配资源”。今天(0609)开始读。只有几十页，每天读一点点也可以很快读完。导读中的“GPA”、“IPO”理解透彻也就理解了本书

重要的是在整体团队形成“透明”的“通用的决策架构”：大家对如何决策达成共识，相当于讨论问题时有了共同共通的背景知识。这样容易形成团队，尤其是当下属知道上级采用怎样的思路框架进行决策时，可以找到讨论的锚点。避免猜测和混乱。

  • 决策的前提：WW——意愿(Will)和能力(able)。where there is a will, there is a way
  • 决策的过程：IPO——信息(Information)、人员(People)、客观推理(Objective Reasoning)。从这三个方面进行决策的讨论：(决策前)哪些信息；(决策中)哪些人员参与以及针对可选方案如何选择
  • 决策的内容：GPA——目标(Goal)、优先级(Priority)、可选方案(Alternatives)。决策必须包含这三个方面，否则不能称为决策，而只是计划、意愿……

时间循环： 先有一个前提，经过一个过程，形成一个结果——按照这个时间顺序形成一个决策。
因果循环： “以终为始，始终服务于最终目标”，后序驱动前序，先有一个目标，促成了一个意愿，评估了能力；然后开始讨论。

WW.IPO.GPA

构成一个“上市公司股票发行”(IPO)“平均分”(GPA: grade point average)的网站(WW)服务。

  • 一级域名：GPA（最终结果——目标，优先级，可选方案）
  • 二级域名：IPO（讨论过程——信息，人员，客观推理）
  • 三级域名：WW（讨论前提——有意愿，有能力）

“开发一个App或小程序”

假设这个目标成立：

  • 有意愿：想做
  • 有能力：能做？如何做？准备投入哪些资源/成本？（人力、时间、金钱）

如何确立这个目标的？

  • 基于哪些信息？
  • 有哪些相关人员参与了讨论？
  • 推理过程是怎样的？

如何实现？

  • 目标是什么？解决哪些问题？
  • 解决这些问题的优先级是怎样排列的？
  • 是否有替代方案？

当上述问题都已明确后，讨论“如何开发一个App或小程序？”

需求定义：一句话描述这个App是做什么的？

  • 有哪些功能？（具体有哪些使用场景？优先级如何排列？）
  • 用户是谁？怎样使用这些功能？为什么他们需要使用这些功能？

开发流程：

  • 采用自研还是外包？
  • 投入的预算？
  • 投入的时间？
  • 投入的人力？

运营：

  • 流量获取、功能策略、统计(需要对应的功能支持……) （到这一步已经很靠后了，但是由一开始定义的“App解决的问题”决定：可以做/如何做不同的运营方式）

顾衡好书榜

两个遗憾：

一是没有选以赛亚·伯林的《浪漫主义的根源》。这么着，在对法国大革命的思想脉络进行梳理的时候，就缺了卢梭这一块。所以就没说透。

另一个缺憾就是没有选拉塞尔·柯克的《保守主义思想》。没有从根子上讲清楚什么是保守主义，保守主义的五大敌人都有什么问题。这么着，对现代性的批判就没直指要害。现在回头看，还是自己喜欢什么就选什么才是最好的策略。
